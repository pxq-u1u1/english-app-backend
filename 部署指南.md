# 阿里云 ECS 部署指南

## 总体架构

```
手机/电脑
  ↓ 打开网页
GitHub Pages（前端）
  ↓ API 调用
阿里云 ECS :3456（后端）
  ↓
DeepSeek API
```

## 第一步：连接服务器

用你阿里云控制台给的公网 IP 和 root 密码，SSH 连上去：

```bash
ssh root@你的服务器IP
```

## 第二步：安装 Node.js

```bash
# 一行命令安装 Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt-get install -y nodejs

# 验证安装
node -v   # 应该显示 v20.x.x
npm -v    # 应该显示 10.x.x
```

## 第三步：上传代码

在你电脑上，把后端文件夹整个上传到服务器。

**方法 A — 用 scp（在电脑终端里跑，不是在服务器上）：**

```bash
# 把 english-app-backend 整个文件夹传到服务器 /root/ 下
scp -r "D:\AAA Github\english-app-backend" root@你的服务器IP:/root/
```

**方法 B — 手动复制：**
1. 在服务器上 `mkdir /root/english-app-backend`
2. 把 `package.json`、`server.js`、`database.js` 三个文件内容分别粘贴进去

## 第四步：启动服务

在服务器上：

```bash
cd /root/english-app-backend
npm install
node server.js sk-你的DeepSeekKey 你的管理密码
```

示例：
```bash
node server.js sk-e6bbf51d51c54dbb8a1f758422038029 english123
```

看到这个就成功了：
```
后端服务已启动: http://0.0.0.0:3456
AI 功能: 已启用
管理密码: english123
```

## 第五步：开放端口

去阿里云控制台 → 安全组 → 添加入方向规则：
- 端口：`3456`
- 来源：`0.0.0.0/0`

不然外面访问不到。

## 第六步：测试

浏览器访问 `http://你的服务器IP:3456/health`

应该返回：
```json
{"ok":true,"time":"2026-05-19T..."}
```

## 第七步：配置前端

1. 打开你的网站 `https://pxq-u1u1.github.io/english-app-react/`
2. 点右上角 ⚙ 设置
3. 填服务器地址：`http://你的服务器IP:3456`
4. 填管理密码：你刚才设的密码
5. 保存
6. 点「同步本地数据到服务器」把电脑上已有的数据传上去

配置完成后，手机打开同一个网址，同样填服务器地址和密码，就能看到同步的数据了。

## 后台运行（保持服务不中断）

关掉终端服务就停了。用 pm2 让它在后台持续运行：

```bash
npm install -g pm2
pm2 start server.js -- "sk-你的Key" "你的密码"
pm2 save
pm2 startup
```

## 日常维护

```bash
pm2 status          # 查看状态
pm2 logs            # 查看日志
pm2 restart server  # 重启
